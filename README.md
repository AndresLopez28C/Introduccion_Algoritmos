## Codigos de algoritmos

Todos los archivos cpp que vez aca son programas en C o C++ los cuales incluyen elementos basicos como captura de datos, mostrar informacion y funciones, todos desarrollados en devC++, solamente estan los archivos en cpp, no los .exe



## funciones 
* C_Parqueadero-> Pide al usuario las horas y minutos de un parqueadero y devuelve el valor a pagar. Programado en C
* Par_o_Impar -> Pide al usuario 1 numero y le dice si es par o impar asi como si es positivo o negativo
* Docenas_exito -> Calcular el valor de un descuento basado en la cantidad de decenas compradas, incompleto
* edades -> Se calcula cuales edades de 3 personas son contemporaneas 
* kilometraje -> Calcula el precio total a pagar por un carro basado en cuantos kilometros recorrio
